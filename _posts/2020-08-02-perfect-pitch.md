闭上眼，哼唱出“Du~”的声音，感受声音在头腔中射出的位置。

| 八度 | 音高 | 位置 |
|:-:|:-:|:-:|
| 2 | G | 嘴唇到下颚间的凹槽 |
| 2 | A | 下嘴唇下部 |
| 2 | B | 下嘴唇中部 |
| 3 | C | 下嘴唇上部 |
| 3 | C#/Db | 上下嘴唇之间 |
| 3 | D | 上嘴唇下部 |
| 3 | D#/Eb | 上嘴唇 |
| 3 | E | 人中 |
| 3 | F | 人中与鼻交界处 |
| 3 | F#/Gb | 鼻孔 |
| 3 | G | 鼻尖 |
| 3 | A | 鼻梁下1/3出 |
| 3 | B | 鼻梁上1/3处 |
| 4 | C | - |
| 4 | D | 睛明穴 |
| 4 | E | - |
| 4 | F | 眉间 |
